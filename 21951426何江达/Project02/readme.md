# 编程作业2：绘制太阳系（只含太阳、月亮、地球）动画

## 要求：

### 必选特性：

1. 光照
   + 太阳为光源
2. 纹理
   + 使用图片进行纹理映射

### 可选特性：

3. 使用顶点着色器和片段着色器
   + 自己实现光照效果
4. 鼠标选择
   + 点击不同的球体显示不同名称

## 我的作业环境：

- Ubuntu 16.04 
- OpenGL 3.3
- Qt 5.13
- ASSIMP 5.0
